# Recast — Brand Identity Study

## Brand Positioning

**Archetype:** The Magician (transformacao) + The Sage (confianca)
**Territorio emocional:** Competencia, calma, empoderamento
**Analogias:** Aesop (premium, privado), Leica (precisao), Muji (simplicidade)

**Semiotic do nome "Recast":**
- Fundicao/metalurgia — derreter e moldar numa nova forma
- Teatro — mesmo conteudo, novo formato
- O prefixo "Re-" — renovacao, iteracao

## Logo Concepts

### Concept A: Wordmark com arrow-A
"Recast" em Syne Bold, barra do "A" substituida por seta (→)
- Conversao embutida na tipografia
- Funciona em todos os tamanhos
- Favicon: "R" num quadrado

### Concept B: Prisma Cromatico
Forma hexagonal/prisma. Feixe entra monocromatico, sai em cores
- Simbolo universal de transformacao
- Favicon: paralelogramo com gradient

### Concept C: Combo — Retangulos Sobrepostos
Dois retangulos a angulos diferentes (input/output) + wordmark
- Antes e depois visualizado
- Favicon: apenas os retangulos sobrepostos

## Brand Voice

**Tom:** Amigo conhecedor que respeita o teu tempo
**Principios:**
- Factos, nao sentimentos ("ficheiros nunca saem do dispositivo" > "preocupamo-nos com privacidade")
- Voz ativa, presente
- Minimal — dizer em menos palavras

**Headlines exemplo:**
- Hero: "Convert anything. Upload nothing."
- Privacidade: "Your files stay on your device. Period."
- Velocidade: "Browser-native speed. No server queue."
- Completo: "Done."
- Filosofia: "Media conversion should be private, fast, and free. So we built that."

**Usar:** convert, transform, recast, private, instant, browser-native, drop, done
**Evitar:** upload, cloud, AI-powered, revolutionary, seamless, robust, please, oops

## Visual Language

| Dimensao | Especificacao |
|----------|--------------|
| Icons | Stroke 1.5px, terminais quadrados, grid 24x24 (Lucide) |
| Borders | 1px foreground/10-15% (subtis) ou full opacity (editorial intencional) |
| Shadows | Quase zero. Usar glows atmosfericos em dark mode |
| Corners | 6-8px (botoes, inputs), 8-12px (cards, modais) |
| Motion | Apenas mudancas de estado, 150-200ms, spring easing |
| Imagens | Nenhuma. Usar: UI real, patterns geometricos, format badges, gradient |

## Recomendacao Final

### Caminho Bold (mais distinto):
- **Paleta:** Opcao C — Chromatic Shift
- **Tipografia:** Sistema C — Syne + Inter + JetBrains Mono
- **Logo:** Wordmark arrow-A + retangulos sobrepostos (favicon)
- **Corners:** 6-8px
- **Shadows:** Glows atmosfericos

### Caminho Safe (mais premium):
- **Paleta:** Opcao A — Warm Foundry
- **Tipografia:** Sistema A — Instrument Serif + DM Sans + IBM Plex Mono
- **Logo:** Wordmark arrow-A
- **Corners:** 0-4px (mais editorial)
- **Shadows:** Minimas
